# 프로젝트명: '에스제이에셋대부(주)' 회사 홈페이지 공지사항 시스템 구축

## 1. 역할 (Role)
너는 시니어 풀스택 개발자이자 UX 디자이너야. 깔끔하고 보안이 철저한 코드를 작성해줘.

## 2. 목표 (Goals)
- 관리자만 접근 가능한 공지사항 작성 페이지 구현
- 일반 유저는 읽기만 가능한 메인 공지사항 목록 구현

## 3. 기술 스택 (Tech Stack)
- Frontend: next.js, Tailwind CSS
- Backend: supabase database

## 4. 핵심 요구사항 (Requirements)
- [ ] 페이지는 메인, 회사소개, 오시는길, 공지사항, 공지사항 상세페이지, 어드민 로그인(/admin) 이렇게 필요해
- [ ] 메인페이지는 히어로 섹션에 회의하고있는 사람들의 사진이 있고 그 위에 회사 이름이 있도록 배치. 바로 아래에 최신 공지사항을 확인할 수 있는 섹션 구현
- [ ] 회사 소개는 대부업체 기본 소개글을 알아서 작성하여 구현
- [ ] 오시는 길은 주소와 카카오 지도가 보여지도록 구현
- [ ] 공지사항은 리스트 형태로 보여지고 클릭하면 상세 페이지가 보이도록 구현. 유저는 댓글이나 반응할 수 없고 확인만 할 수 있도록
- [ ] `/admin` 경로로 접속했을 때만 로그인 폼 노출
- [ ] 로그인 성공 시에만 공지사항 페이지에서 작성, 수정, 삭제 버튼 활성화

## 5. 제약 사항 (Constraints)
- 별도의 백엔드 서버 없이 supabase만 사용할 것
- 디자인은 최대한 미니멀하고 화이트 톤으로 유지할 것
